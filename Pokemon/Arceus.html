Arceus

Arceus is a normal type pokemon. This is a legendary pokemon which has a 1.6 percent catch rate it can change type whether its holding a plate or type specific z crystal. This pokemon closely resembles a qilin or centaur with a gray body. Arceus is believed to have created the whole pokemon world. This pokemon is so powerful it is said to have the power of bringing back things that it once destroyed.

Battle advantages and disadvantages

Arceus is a normal type pokemon so it does not have an advantage on anyone but the only battle disadvantage it has is against fighting type pokemon
